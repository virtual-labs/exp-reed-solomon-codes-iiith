2     Basics of MDS Codes

In this section, we define MDS codes and list a few important properties of these codes. MDS codes are a class of error-correcting codes that can correct a maximum number of errors for the given code rate (k/n). For any given code C(n,k)\mathcal{C}(n,k)C(n,k), the upper bound on the minimum distance dmind_{min}dmin​ of the code is called a Singleton bound, which is as follows: dmin≤n−k+1. \begin{equation} d_{min} \leq n-k+1. \end{equation} dmin​≤n−k+1.​​ Note that error-correcting capability depends upon the dmind_{min}dmin​, since the number of errors that the code can correct is equal to ⌊dmin−12⌋\lfloor \frac{d_{min} -1}{2} \rfloor⌊2dmin​−1​⌋ . So, the formal definition of MDS code is defined as follows:

Definition 2(MDS Code). A code C(n,k)\mathcal{C}(n,k)C(n,k) is called MDS code if the minimum distance achieves the Singleton bound (1), i.e., dmin=n−k+1d_{min}=n-k+1dmin​=n−k+1.

Now, We will describe a few properties of MDS codes without proof.

  • A code C(n,k)\mathcal{C}(n,k)C(n,k) is MDS, if and only if every set of n−kn-kn−k columns of its parity check matrix HHH are linearly independent.
  • Dual code of an MDS code is MDS.
  • Every set of kkk columns of its generator matrix GGG are linearly independent.
